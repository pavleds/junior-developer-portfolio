# Apex Garage

A fictional performance-car marketplace built as a front-end portfolio project with HTML, CSS and JavaScript.

## Features

- Responsive automotive marketplace interface
- Inventory search and multi-field filtering
- Browser-persisted favourites
- Two-vehicle specification comparison
- Interactive illustrative finance calculator
- Vehicle details and validated enquiry flow
- Clear demonstration-data notices

No vehicles, prices, finance offers or enquiries are real. The project has no production backend, user accounts or payment processing.
