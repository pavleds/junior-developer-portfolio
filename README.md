# Junior Developer Portfolio

A static, publishable portfolio containing a professional homepage, two working applications, detailed case studies, and a beginner-friendly learning guide.

## Start

Open `index.html` in a modern browser. No installation is required.

## Projects

- **Banja Bus Station:** bilingual departure search, time filters, statuses, and responsive public information.
- **Kafa Café Operations:** register, kitchen workflow, menu availability, reporting, CSV export, localization, and browser persistence.


## Boundaries

- Station information is fictional demonstration data.
- The café app is not a fiscal register or payment system.
- Neither demo has a backend, accounts, or multi-device synchronization.
- Understand and personalize the work before presenting it professionally.
